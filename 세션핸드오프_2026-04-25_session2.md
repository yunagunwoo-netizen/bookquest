## A. 오늘 변경 요약

### 1. 배틀 카드 UI — 포켓몬카드 스타일 전면 리디자인
- **BattleCard** (배틀 진행 중): 기존 단순 아바타+HP바 → 포켓몬카드 스타일로 변경
  - 상단바: 타입 아이콘 + 이름 + HP 숫자 (레어리티 색상)
  - 아바타 영역 (110px): 레벨 뱃지 + 레어리티 뱃지
  - 칭호 표시 (getTitle)
  - 기술 2개 표시 (레어리티별 다른 기술명/데미지)
  - HP 바 + 타입 라벨
  - 샤인 오버레이 애니메이션
  - cardImg 로드 실패 시 iconImg/img 폴백
- **ResultCard** (배틀 결과): 동일한 포켓몬카드 스타일로 재디자인 (130px 축소판)
  - 승자에게 왕관 표시
- **CARD_TYPE / CARD_TYPE_ICON / CARD_SKILL** 상수 추가: 레어리티별 타입명/아이콘/기술 정의
- **RARITY_BADGE_STYLE / RARITY_LABEL** 에 heroic, mythic, hidden 추가

### 2. AI HUB 아바타 랜덤 대사
- 기존: 완독 수에 따라 고정 3개 대사
- 변경: 3단계 x 8개 = 총 24개 대사가 접속 시마다 랜덤 표시
  - 0권: 첫 방문 친구 환영
  - 1~2권: 함께 읽은 경험 언급
  - 3권+: 칭찬/응원/레벨업

### 3. 카드 이미지 시스템 (이전 세션 연장)
- 52개 카드 PNG (600x840) 생성 완료 → `avatars/card/` 디렉토리
- Python Pillow 합성: 그라데이션 배경 + 레어리티 테두리 + 아바타 오버레이
- `cardImg` 필드 AVATAR_DATA에 자동 주입 (line 1901-1903)

### 4. 기타 버그 수정
- `rounds[0]` → `result.rounds[0]` (배틀 크래시 버그)
- 책 상세 탭 버튼 가로 스크롤 → flexWrap 적용

## F. 주의사항 (오늘 겪은 버그와 해결법)

### 1. rounds[0] 참조 오류
- **증상**: 배틀 API 응답 후 크래시
- **원인**: `const firstRound = rounds[0]` — `rounds` 변수가 undefined
- **해결**: `const firstRound = result.rounds[0]` 로 수정

### 2. cardImg 폴백 미작동
- **증상**: 카드 이미지 없는 캐릭터(브롤 등)에서 체크무늬 깨진 이미지
- **해결**: BattleCard에 `onError` 핸들러 추가, `avatarSrc.replace("avatars/card/", "avatars/").replace(".png", "_sm.png")` 로 폴백

### 3. CARD_TYPE/CARD_SKILL 스코프 문제
- **증상**: interactive IIFE 내 정의된 상수를 result IIFE에서 참조 불가
- **해결**: result 블록에 `_R` 접미사로 별도 상수 선언 (CARD_TYPE_R, CARD_SKILL_R 등)

### 4. 카드 이미지 생성 타임아웃
- **증상**: 52개 카드 합성 시 vignette 효과(pixel-by-pixel numpy)로 45초 타임아웃
- **해결**: vignette/shine 제거, 단순 line-by-line gradient + draw.rounded_rectangle 사용, 5배치 분할

### 5. 주석 처리 아바타 재발 방지
- 과거 sejong/arthur/zeus가 주석(//)에 묻혀 파싱 안 되는 버그 있었음
- 오늘 검증: 주석 처리된 엔트리 0개 확인
