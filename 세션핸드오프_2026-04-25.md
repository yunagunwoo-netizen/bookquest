## F. 주의사항 (오늘 겪은 버그와 해결법)

### 1. 이미지 투명 처리 시 하단 잘림 버그
- **증상**: Python Pillow로 투명 처리 시 `for y in range(int(h*0.5), h)` 코드가 하단 절반의 near-black 픽셀을 제거 → 캐릭터 몸체 잘림
- **영향**: hypatia, galileo, musashi 이모티콘 파손
- **해결**: 배경색만 제거하는 단순 로직으로 재처리. 사용자 판단: "이미지 편집은 클로드에게 시키지 않겠다"
- **교훈**: 이미지 편집은 절대 임의 픽셀 제거 로직 사용 금지. 배경색 exact match만 허용.

### 2. 체커보드 패턴이 실제 픽셀로 포함된 아이콘
- **증상**: AI 생성 아이콘에 투명 배경 대신 체커보드 패턴이 RGB 픽셀로 베이킹됨
- **해결**: 프롬프트에 "solid color background (not checkerboard)" 명시 후 재생성, 사용자가 별도 도구로 투명 처리

### 3. PowerShell 커밋 메시지 파싱 오류
- **증상**: 한글+특수문자 포함 커밋 메시지를 여러 줄로 입력 시 PowerShell이 명령어로 해석
- **해결**: `git commit -m "한줄메시지"` 형태로 단일 라인 사용, 또는 GitHub Desktop 사용

### 4. Enchanted Abyss 스킨 적용 범위
- **배틀 화면(4500~4900)은 기존 다크 보라 유지** — 별도 수정 안 함
- `background: "white"` 6건 잔존: 모두 배틀 화면 버튼(의도적) 또는 프로그레스바 fill
- 색상 변경 시 반드시 `color: "white"` 가 다크 배경 위인지 확인 (밝은 배경 위 white → 안 보임)
