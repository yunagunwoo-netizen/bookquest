- **현상**: Edit 대규모 적용 후 파일이 JSX 중간에서 끊겨 있음 (16 KB 손실)
- **탐지**: `</body>`/`</html>`/`root.render` 이 파일 끝에 없음
- **해결**: 직전 백업(`index_pre_v34_backup.html`)에서 공통 anchor(마지막 80 bytes)를 찾아 그 뒤를 그대로 append
- **재발 방지**:
  ```bash
  tail -c 200 index.html   # 꼬리에 </html> 확인
  grep -c '<script' index.html && grep -c '</script>' index.html   # 짝 수 일치 확인
  ```
  매 큰 변경 뒤 이 두 커맨드로 1초 점검
