## 🎬 아이디어 #001 — 오리지널 캐릭터 유튜브 숏츠 마케팅

**등록일:** 2026-04-22
**분류:** 마케팅 · 콘텐츠 · 리텐션
**상태:** 💡 아이디어 (미착수)

### 배경
- 마법천자문은 "학습 만화"로 아이들에게 재미있는 학습 동기를 만든 대표 사례.
- 마법천자문은 **책 판매**가 주 사업이지만, BookQuest의 목표는 **앱 이용 유도**.
- AI Coach 훈련 드릴용 10초 애니메이션을 **힉스필드(Higgsfield)** 로 만들어본 경험 있음 → 제작 파이프라인 이미 검증됨.

### 아이디어 요약
BookQuest 오리지널 캐릭터(루미·미오 + 플레이어 아바타들)를 주인공으로 한 **30초~1분 짜리 애니메이션 여러 편**을 힉스필드로 제작해 **유튜브 숏츠**에 배포.

### 핵심 목표
- 아이들이 "독서가 재미있다"고 느끼게 만드는 감정적 훅(hook) 제공
- 브랜드·캐릭터 인지도 확보 → 앱 설치/재방문 유도
- 부모 커뮤니티 바이럴 (숏츠는 부모가 먼저 발견해서 아이에게 보여줌)

### 포맷 아이디어 (시드)
1. **미니 에피소드**: 루미·미오가 책 속 세계로 빨려 들어가 문제를 푸는 30초 단편
2. **Q&A 배틀 하이라이트**: 아바타 배틀 리플레이 시스템과 연계된 "명장면" 컷
3. **아바타 진화 순간**: 진화 시 힉스필드 연출(Task #7) 에서 파생 → 숏츠 소재 재활용
4. **책 소개 숏츠**: 주요 도서 한 권당 "이 책을 읽으면 이런 일이" 10초 티저
5. **캐릭터 대화**: 루미·미오 둘의 짧은 티키타카 만담 (챕터 예고편)

### 예상 제작 규모
- 1편당 제작 시간: 힉스필드 생성 + 편집 ≈ 2~4시간 (경험치 고려)
- 1차 배포 목표: **10편** (2~3주 내)
- 주 업로드: 2~3편 (알고리즘 유입)

### 비용 추정 (러프)
- 힉스필드 구독료: 월 USD 기준 (추후 별도 산정)
- 효과음·배경음악: CC0 또는 저작권 무료 소스 + AI 생성
- 편집 인력: 본인 (마케팅 경력 활용)

### 측정 지표
- 숏츠 조회수·저장·공유
- 채널 구독자 증가율
- **앱 신규 설치 수** ← 가장 중요 (UTM 링크 또는 앱스토어 레퍼러)
- 코멘트 분석 (타깃 연령대 실제로 보는지)

### 시너지
- Task #7 (아바타 진화 힉스필드 연출) 결과물을 숏츠 소스로 재활용 가능
- Phase 5b Q&A 배틀 리플레이 시스템 → 자동으로 "명장면 숏츠" 재료 생산
- 분기 시즌 TOP10 도서 선물 발표도 숏츠 콘텐츠화 가능 (수상자 축하 영상)

### 다음 액션 (착수 시)
1. 캐릭터 스타일 시트 정리 (힉스필드 일관성용 레퍼런스 이미지)
2. 첫 1편 파일럿 제작 → 반응 테스트
3. 10편 스토리보드 일괄 기획 → 파이프라인 효율화
4. 유튜브 채널 생성·브랜딩

## 📚 아이디어 #002 — 교과 연계 도서 학년별 권수 2배 확장

**등록일:** 2026-04-29
**분류:** 콘텐츠 · 데이터 확장
**상태:** 💡 아이디어 (미착수, 우선순위 중간)

### 배경
- 현재 CURRICULUM_BOOKS는 학년별 6권 (총 9학년 × 6권 = 54권 구성).
- 한 학기 국어 단원이 8~10개인데 책이 6권뿐이라 단원-책 매핑에서 같은 책이 여러 단원에 반복 등장.
- 2026-04-29 단원 매칭 시스템을 "단원 → 책"에서 "책 → 단원 태그"로 구조 전환하면서 책 권수가 추천 다양성의 핵심 자원이 됨.

### 목표
- 학년별 6권 → **12권**으로 2배 확장 (총 54 → 108권).
- 단원별 매칭 다양성 확보 + 학년 안에서 학생이 더 많은 선택지 확보.

### 작업 단위
- 학년별 추가 6권씩 책 큐레이션 (교과서 추천도서 / 권장도서 풀 활용).
- 각 책마다 메타데이터 + curriculumLinks + 토론 25 / 퀴즈 25 / 심화 6~15 풀.
- Phase B 풀 25/25/8 확장 패턴 그대로 적용.

### 시너지
- 추가된 책마다 같은 톤 가이드 적용 → 재사용 가능
- "이 책은 어떤 단원에 매칭되나" 화면에서 책 권수 늘면 단원 카버리지도 자연 증가
- 시즌 도서·교과 연계 도서 풀 통합으로 추후 추천 알고리즘 풍성해짐

### 다음 액션 (착수 시)
1. 학년별 추가 6권 후보 큐레이션 (학교 추천도서 + 권장도서 매칭)
2. 책 메타데이터 작성 (id C055 ~ 부터 또는 별도 prefix 검토)
3. curriculumLinks 매핑 (각 책이 어느 학기 어느 단원에 적합한지)
4. 풀 25/25/8 작성 — Phase B 톤 가이드 적용
5. CurriculumScreen 책 카드 리스트가 자동으로 12권 표시
