## E. 알려진 이슈 / 주의사항

1. **PWA 설치**: `beforeinstallprompt`는 HTTPS + manifest + SW가 모두 갖춰져야 Chrome이 발화. 로컬 테스트 시 localhost에서만 동작.
2. **복구 검색**: `findUsersForRecovery`는 Firestore 전체 컬렉션을 `get()`으로 읽음 — 사용자 수가 많아지면 쿼리 최적화 필요 (이름+학교 필드로 `where` 쿼리).
3. **자동 백업**: 앱이 열릴 때 1일 1회 실행. 앱을 안 열면 백업 안 됨.
4. **V34_HOF_VISIBLE_IDS**: `V35_ICON_IDS.filter(id => !HIDDEN_HERO_IDS.includes(id))`로 동적 생성. HIDDEN_HERO_IDS가 현재 빈 배열이므로 V35_ICON_IDS 32종 전체가 visible.
